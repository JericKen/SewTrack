# 🧵 SewTrack

SewTrack is a web-based management system designed for tailoring and textile stores. It helps manage inventory, repair orders, sales, expenses, and business reports in one centralized platform.

## ✨ Features

- 📦 Inventory Management
- 🏷 Category Management
- 🧵 Textile Product Management
- ✂ Repair Order Tracking
- 💰 Sales Recording
- 📉 Expense Tracking
- 📊 Dashboard & Reports

## 🛠 Tech Stack

### Frontend
- React
- Vite

### Backend
- Node.js
- Express.js

### Database
- PostgreSQL
- Prisma ORM
