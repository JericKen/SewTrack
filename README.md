# SewTrack

SewTrack is a web-based management system for tailoring and textile shops.

## Features

- Inventory Management
- Category Management
- Product Management
- Repair Order Tracking
- Sales Recording
- Expense Tracking
- Dashboard & Reports

## Tech Stack

### Frontend
- React
- Vite

### Backend
- Node.js
- Express.js

### Database
- PostgreSQL
- Prisma ORM
